data TableConstraint Source #

Type to represent a table constraint that would be part of a CREATE TABLE or ALTER TABLE statement. For instances, the UNIQUE constraint in

CREATE TABLE FOO
 ( id integer
 , UNIQUE id
 )

TableConstraint provides a SqlExpression instance. See unsafeSqlExpression for how to construct a value with your own custom SQL.
