Orville.PostgreSQL.Execution

Contents

  • High level modules for most common tasks
  • Mid-level modules with more flexibility for handling less common queries
  • Low-level modules for executing and decoding SQL expressions yourself

Description

You can import Orville.PostgreSQL.Execution to get access to all the execution-related functions and types exported by the modules below. This includes a number of lowel-level items not exported by Orville.PostgreSQL that give you more control (and therefore responsibility) over how the SQL is executed.

Since: 1.0.0.0
